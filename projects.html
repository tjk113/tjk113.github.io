# Projects

## GemSharp - a basic terminal Gemini client

I've written about Gemini before, so I won't reiterate my interest in the protocol here. I decided to use C# for this project at the suggestion of a friend, and have found it to be quite a nice language. Being tied into the Microsoft ecosystem doesn't bother me too much, as I'm primarily a Windows user anyway (and am familiar with the occasional horror that is Visual Studio). Gemini is a simple protocol by design, so the networking code was quite straightforward. Hand-writing a TUI, on the other hand, has been yet another enlightening experience, to say the least.

## tilda - a C-like general purpose programming language

I enjoy programming in a variety of languages (my GitHub serves as evidence). Couple that with an innate interest in understanding what's going on "behind the scenes," and I was bound to try writing my own programming language sooner or later. I've been following the book Crafting Interpreters by Robert Nystrom, which has been a fantastic guide through language development. I've gained a lot of C++ experience in writing my interpreter (for better or worse), because I'm not a fan of the book's implementation language of Java. One day I'd like to write a native compiler for it (or another language).
